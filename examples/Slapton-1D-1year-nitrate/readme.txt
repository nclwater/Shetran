Slapton Wood - water flow, contaminant transport and nitrate example

Uses rundata_slap.txt
One year simulation. 1D as it uses 1 grid square with no rivers and a base flow boundary condition.

Rainfall is 2mm/day execept for 10 days near the start with no rainfall. PE is 1mm/day

note: the data CM57-61 in the cmd is currently read in but not used. The values are hardcoded to 0.5
